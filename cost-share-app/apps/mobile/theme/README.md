# Theme Directory

This directory contains theme configuration and styling constants.

## Purpose

Centralize all theme-related configuration:
- Colors
- Typography
- Spacing
- Border radius
- Shadows

## Future Implementation

Create files like:
- `colors.ts` - Color palette
- `typography.ts` - Font sizes and weights
- `spacing.ts` - Spacing scale
- `index.ts` - Export all theme values

## Usage

```typescript
import { colors, spacing } from './theme';
```

This ensures consistent styling across the entire app.
