# Components Directory

This directory contains reusable UI components.

## Purpose

All reusable UI elements should be placed here to promote:
- Code reusability
- Consistent UI across the app
- Easier maintenance
- Better testing

## Examples of components to add:

- `Button.tsx` - Custom button component
- `Input.tsx` - Custom input field
- `Card.tsx` - Card container
- `Loading.tsx` - Loading indicator
- `EmptyState.tsx` - Empty state component
- `GroupCard.tsx` - Group display card
- `ExpenseCard.tsx` - Expense display card

## Rules

- Components should be pure and reusable
- No business logic in components
- Use TypeScript for all components
- Style with NativeWind (Tailwind classes)
- Export components from index.ts for clean imports
